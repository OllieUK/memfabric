# WP-077 Plan: Extract shared schema-init utilities + fix embeddings model routing

## Summary

Two independent housekeeping tasks that are prerequisites for all knowledge-layer WPs:

1. **Extract shared schema-init utilities** — `create_constraint()` and `get_embedding_dimension()` are byte-identical in `scripts/init_schema.py` and `scripts/init_knowledge_schema.py`. Extract to `scripts/schema_utils.py`; both scripts import from there.

2. **Fix `embeddings.py` multi-model support** — `memory_service/embeddings.py` uses a module-level singleton locked to `EMBEDDING_MODEL`. `migrate_embeddings.py` reads `settings.knowledge_embedding_model` but calls `get_embedding()` without passing the model name, embedding knowledge nodes with the *wrong* model. Fix by adding an optional `model_name` parameter to `get_model()`, `get_embedding()`, and `get_embedding_dimension()`, backed by a `_model_cache` dict. All existing callers pass no argument → unchanged behaviour.

Both tasks are sequentially executed by a single Sonnet agent (WP-077 is not team-parallelised — files are small and both tasks complete quickly).

```python
_model_cache: dict[str, SentenceTransformer] = {}

def _load_model_by_name(model_name: str) -> SentenceTransformer:
    """Load a SentenceTransformer by explicit name, with same offline handling as _load_model."""
    kwargs = {}
    if _local_files_only():
        os.environ.setdefault("HF_HUB_OFFLINE", "1")
        os.environ.setdefault("TRANSFORMERS_OFFLINE", "1")
        kwargs["local_files_only"] = True
    try:
        return SentenceTransformer(model_name, **kwargs)
    except Exception as exc:
        if kwargs.get("local_files_only"):
            raise RuntimeError(
                f"Could not load the embedding model '{model_name}' from local files. "
                "Cache the model locally first or set EMBEDDING_LOCAL_FILES_ONLY=false."
            ) from exc
        raise RuntimeError(f"Could not load the embedding model '{model_name}'.") from exc
```

**Step 2b — Modify `get_model`:**
```python
def get_model(model_name: str | None = None) -> SentenceTransformer:
    global _model
    if model_name is None or model_name == _model_name:
        # Singleton path — backward-compatible
        if _model is None:
            _model = _load_model()
        return _model
    # Named model path — use cache
    if model_name not in _model_cache:
        _model_cache[model_name] = _load_model_by_name(model_name)
    return _model_cache[model_name]
```

```python
def _cache_key(text: str, model_name: str | None = None) -> str:
    """Return a SHA-256 hex digest for the given model+text pair."""
    effective_model = model_name if model_name is not None else _model_name
    raw = f"{effective_model}:{text}"
    return hashlib.sha256(raw.encode("utf-8")).hexdigest()
```

**Step 2e — Modify `get_embedding`:**
```python
def get_embedding(text: str, model_name: str | None = None) -> list[float]:
    """Return an embedding vector for *text* using the specified model.

    model_name=None → uses EMBEDDING_MODEL (default, backward-compatible).
    model_name=<name> → uses that model (loaded/cached on first call).

    Disk cache is keyed by model+text — different models produce separate entries.
    """
    if _cache_dir:
        cache_path = Path(_cache_dir) / f"{_cache_key(text, model_name)}.json"
        if cache_path.exists():
            return json.loads(cache_path.read_text(encoding="utf-8"))

        embedding: list[float] = get_model(model_name).encode(text).tolist()
        cache_path.parent.mkdir(parents=True, exist_ok=True)
        cache_path.write_text(json.dumps(embedding), encoding="utf-8")
        return embedding

    return get_model(model_name).encode(text).tolist()
```
